# 条件
+ Microsoft365の最新バージョンを利用する
+ Excel in Pythonが動くか確認、CSVが表示されない場合は対応してない場合があります
+ 救急・小児科・島しょ部はそれぞれ期間が違うため別々に入力
+ kindはソートのため必要、今後変更の可能性があるため連番にしていない
+ もし追加の場合は適切な位置に配置する。

# 共通
1. "救急病院データ作成2026.xlsx"をダウンロード
2. 祝日を取得ため外部アクセスを許可にする https://www8.cao.go.jp/chosei/shukujitsu/syukujitsu.csvから祝日を取得している

# 「救急」「特別」シートの入力手順
1. 「救急」のシートの基準日（月初めの1日）を指定
2. 「救急」のシート医師会からのFAXより各病院のマークを読み取り病院名の行に日にちを入力
3. 入力が完了したら「救急マーカー確認」のシートでマークの位置を確認
4. 「特別」のシートに備考内の病院を入力、不要な場合はday列の日にちを削除しておくと除外される
5. 「救急」と「特別」のデータが重複していた場合は「特別」の方で上書き
6. 「救急CSV」をファイル保存でCSVで保存する

# 「小児科」シートの入力手順
1. 各病院の日付を読み取り病院名の列に日付を入力
2. 「小児科CSV」をファイル保存でCSVで保存する
3. 「内科CSV」をファイル保存でCSVで保存する　※日曜日の小児科と同日のため生成

# 「島しょ部」シートの入力手順
1. 各病院の日付を読み取り病院名の列に日付を入力
2. 「島しょ部CSV」をファイル保存でCSVで保存する
